# Vicsek-Loner Showcase 中文说明

[English Version](./README.md)

这是一个面向 GitHub 展示的轻量级可视化仓库，用来演示本项目中 `Vicsek + 演化博弈 + loner` 模型在小规模情形下的动态演化。

它不是为了替代 MATLAB 科研代码，而是为了回答一个更直观的问题：

`这些智能体到底是怎么动起来的？`

在这个展示版里，你可以直接看到：

- 智能体如何在二维周期边界空间中运动
- 半径 `r` 如何决定邻居关系
- 合作者和 loner 如何跟随局部平均方向
- 背叛者如何随机改变方向
- 策略如何通过 Fermi 模仿规则更新
- loner 如何通过额外因子 `p` 调整模仿概率

默认设置是 `10` 个智能体、`20` 轮演化，目的是让人能够直接看懂过程。

## 页面里有什么

首页主要分成三部分：

1. 顶部说明区  
   简要介绍模型，并放一段可直接观看的演示视频。

2. 空间动画区  
   展示每个智能体的位置、运动方向以及交互半径。

3. 指标图表区  
   展示三种策略频率随时间的变化，以及全局 `Va` 和合作水平的时间序列。

这样做的目的，是把“微观怎么动”和“宏观指标怎么变”放在同一个页面里，方便理解。

## 与 MATLAB 代码的对应关系

这个前端展示版不是随意编的动画，而是尽量和当前 MATLAB 实现保持一致。它主要参考这些文件中的规则：

- `simulation_loner.m`
- `is_neighbour.m`
- `neighbour_to_imitate.m`

前端展示版保留了这些核心逻辑：

1. 位置更新  
   智能体以固定速度 `v0` 在正方形周期边界中移动。

2. 邻居判定  
   两个智能体的周期距离小于 `r` 时，视为邻居。

3. 方向更新  
   - `C` 和 `L`：沿局部平均方向前进，并叠加噪声 `eta`
   - `D`：随机选择新的方向

4. 收益更新  
   - 利用局部一致性得到局部 `Va_i`
   - 通信带来代价
   - 收益定义为 `Va_i - alpha * cost`

5. 策略更新  
   每个智能体随机选一个邻居，用 Fermi 规则决定是否模仿；
   对于 loner，再额外乘以 `p`

因此，这个展示版更适合做“机制解释”和“项目展示”，而真正的科研量化结果仍然应以 MATLAB 仿真为准。

## 如何使用

直接打开 `index.html` 就可以运行。

页面控件允许你调整：

- 智能体数量
- 演化轮数
- 半径 `r`
- loner 因子 `p`
- 相对代价 `alpha`
- 噪声 `eta`
- 速度 `v0`
- 随机种子

按钮功能如下：

- `Reset`：按当前参数重新生成初始状态
- `Play`：播放动画
- `Step`：向前推进一轮
- `Run 20 Rounds`：直接跑完当前设定轮数
